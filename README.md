# Kilo-cd

Kısa açıklama

- Kilo-cd, Kilo for VS Code eklentisi için bir CD/CI yardımcı aracı veya bu projeye ait dağıtım/alt yapı komutlarını otomatikleştirmeyi hedefleyen bir araçtır. (Buraya proje amacınızı kısa ve net bir şekilde yazın.)

Durum

- Durum: Geliştirme

Gereksinimler

- Node.js >= 16 (varsa)
- VS Code (eğer bir VS Code eklentisi ise)
- Diğer bağımlılıklar: (ör. npm, yarn vb.)

Kurulum (yerel)

1. Repo'yu klonlayın:
   git clone https://github.com/emrahtoraman-rgb/Kilo-cd.git
2. Klasöre geçin:
   cd Kilo-cd
3. Bağımlılıkları yükleyin (varsa):
   npm install
   veya
   yarn

Çalıştırma / Test

- Proje türüne göre buraya çalıştırma ya da test komutlarını ekleyin. Örnek:
  - VS Code eklentisi: `code --install-extension ./kilo-cd-0.0.1.vsix` veya F5 ile debug
  - Node uygulaması: `npm start` veya `node index.js`

Kullanım

- Kısa kullanım örnekleri veya komutlar. (Ör: eklenti paletinden "Kilo: Deploy" komutu ile...)
